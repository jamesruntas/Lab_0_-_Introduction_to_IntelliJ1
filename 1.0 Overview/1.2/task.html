~1.0 Overview~

You should use this lab to familiarize yourself with the IntelliJ EduTools environment. It is the environment in which labs will be delivered. EduTools allows for feedback and help with your solution. A walkthrough of this lab is posted on CuLearn.

~Task 1.2~

Other tasks will require you to validate your code against the correct output.

In this task, you are being asked to print out "OOP!". You should edit the required files by adding your code in.

Every task has its own isolated src folder. Although it is not visible to you within EduTools, it is there. You can add and remove files from this folder by right clicking the task and selecting a new Java class (1.2>New>Java Class).

Main.java is the point from which our code is run, because it contains the main() method. This is all you need to understand for now, but there will be more to come soon.

You can click on 'Check' and then on 'Compare Outputs' to bring up a window that you can use to debug your code.

Run it!
